# Todo Kita

1. run "Flutter pub get"

- lib
  - database
    - database_helper.dart
  - model
    - constants.dart
    - task_model.dart
  - ui
    - categories.dart
    - todo_list_screen.dart
  - widget
    - task_card.dart
  - main.dart
